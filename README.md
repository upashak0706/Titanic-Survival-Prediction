The project focuses on building and evaluating multiple machine learning classification models to predict the survival of passengers on the Titanic based on demographic and trip-related data.
1. Dataset and Data CleaningThe analysis uses the classic "Titanic" dataset loaded via the Seaborn library.Feature Selection: The project initially identifies 15 columns, including survived, pclass, sex, age, and fare.Data Pruning: Several columns such as deck, embark_town, alive, and class are dropped to streamline the feature set.Handling Missing Values: Missing age values are imputed with the mean, and rows with missing embarked data are removed.
2. Data PreprocessingTo prepare the data for classification algorithms, several transformations are applied:Label Encoding: Categorical features like sex and embarked are converted into numerical values.Feature Scaling: For algorithms sensitive to feature magnitude (like KNN and SVM), a StandardScaler is used to normalize the data.Data Splitting: The dataset is split into training (80%) and testing (20%) sets to validate model performance.
3. Model Implementation and ComparisonThe notebook implements and evaluates five different classification algorithms:Logistic Regression: Achieved an accuracy of approximately 80.3%.K-Nearest Neighbors (KNN): Using $k=5$, this model achieved an accuracy of 77.5%.Gaussian Naive Bayes: Also yielded an accuracy of 77.5%.Decision Tree Classifier: Resulted in an accuracy of 76.9%.Support Vector Machine (SVM): Using a linear kernel, this model matched the top performance with an accuracy of 80.3%.
4. Model Evaluation and ValidationDetailed Metrics: For each model, the project generates a confusion matrix and a classification report providing precision, recall, and F1-score for both survival outcomes.Cross-Validation: To ensure the stability of the results, the SVM model underwent 5-fold cross-validation, resulting in a mean accuracy score of 78.7%.ConclusionThe project demonstrates that Logistic Regression and Support Vector Machines are the most effective models for this dataset, both achieving the highest baseline accuracy of 80.3%.
